JavaScript Basics

Variables

Definition: Variables are used to store data values.

let name = 'John'; 
const age = 30;

Functions

Definition: Reusable blocks of code used to perform a specific task.


function greet() { 
    return 'Hello!'; 
}

Events

Definition: React to user actions like clicks or keypresses.


document.getElementById('btn').addEventListener('click', () => alert('Clicked!'));

DOM Manipulation

Definition: Modify HTML content dynamically.


document.getElementById('myDiv').innerText = 'Updated!';

Arrays

Definition: Store multiple values in a single variable.


let numbers = [1, 2, 3];

Objects

Definition: Collections of key-value pairs.


let person = { 
    name: 'John', 
    age: 30 
};

Loops

Definition: Execute code repeatedly.


for (let i = 0; i < 5; i++) { 
    console.log(i); 
}

Promises

Definition: Handle asynchronous operations.


fetch('url')
    .then(response => response.json())
    .then(data => console.log(data));

Classes

Definition: Templates for creating objects.


class Car { 
    constructor(brand) { 
        this.brand = brand; 
    } 
}

Local Storage

Definition: Store data locally in the browser.


localStorage.setItem('key', 'value');
