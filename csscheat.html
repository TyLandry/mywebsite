CSS Features

Feature Definition Example
Selectors Used to target HTML elements for styling. 
p { color: red; }
Classes and IDs Target specific elements. 
.class-name { font-size: 16px; }
#id-name { margin: 10px; }
Flexbox A layout model for aligning elements. 
div { display: flex; justify-content: center; }
Grid A two-dimensional layout system. 
div { display: grid; grid-template-columns: 1fr 2fr; }
Media Queries Used for responsive design. 
@media (max-width: 600px) { body { background-color: lightblue; } }
Pseudo-classes Apply styles to elements based on their state. 
a:hover { color: green; }
Transitions Enable animations between states. 
button { transition: background-color 0.3s; }
Positioning Specifies the positioning of elements. 
div { position: absolute; top: 50px; left: 100px; }
Box Model Defines how padding, borders, and margins affect an element's size. 
div { margin: 10px; padding: 5px; border: 1px solid black; }
Variables Custom properties for reusable values. 
:root { --main-color: #ff5733; }
Hover Effects Apply styles when the user hovers over an element. 
button:hover { background-color: blue; }
Background Properties Set background colors, images, and gradients. 
div { background: linear-gradient(to right, red, yellow); }
Clip Path Defines a clipping region for an element. 
div { clip-path: circle(50%); }
Filter Applies graphical effects like blur or grayscale. 
img { filter: grayscale(100%); }
Aspect Ratio Defines a preferred aspect ratio for an element. 
div { aspect-ratio: 16 / 9; }
Content Overflow Handles content that overflows its container. 
div { overflow: hidden; }
Custom Fonts Use external font resources. 
@font-face { font-family: 'MyFont'; src: url('myfont.woff2'); }
Cursor Property Changes the appearance of the cursor. 
a { cursor: pointer; }
Outline Styles the outline of an element. 
button { outline: 2px solid red; }
Alignment Aligns text or elements. 
p { text-align: center; }
