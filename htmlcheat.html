Learn About HTML Elements

  • <html>: The root element of an HTML document.
    <html><head></head><body></body></html>
  • <head>: Contains metadata about the document.
    <head><title>Page Title</title></head>
  • <title>: Sets the title of the document.
    <title>My Awesome Page</title>
  • <body>: Contains the content of the document.
    <body><h1>Welcome!</h1></body>
  • <h1> - <h6>: Represents headings of different levels.
    <h1>Main Heading</h1><h2>Subheading</h2>
  • <p>: Represents a paragraph of text.
    <p>This is a paragraph.</p>
  • <a>: Defines a hyperlink.
    <a href="https://example.com">Visit Example</a>
  • <img>: Embeds an image.
    <img src="image.jpg" alt="Description">
  • <ul>: Creates an unordered list.
    <ul><li>Item 1</li><li>Item 2</li></ul>
  • <ol>: Creates an ordered list.
    <ol><li>First item</li><li>Second item</li></ol>
  • <li>: Represents an item in a list.
    <li>List item</li>
  • <div>: Defines a block-level container.
    <div><p>Content inside a div.</p></div>
  • <span>: Defines an inline container.
    <p>This is a <span style="color: red;">red</span> word.</p>
  • <input>: Creates an input field.
    <input type="text" placeholder="Enter your name">
  • <form>: Defines a form for user input.
    <form action="/submit"><input type="text" name="username"><button type="submit">Submit</button></form>
  • <button>: Creates a clickable button.
    <button>Click Me</button>
  • <table>: Defines a table structure.
    <table><tr><th>Name</th><th>Age</th></tr><tr><td>Alice</td><td>30</td></tr></table>
  • <th>: Defines a header cell in a table.
    <th>Column Name</th>
  • <td>: Defines a cell in a table.
    <td>Table Data</td>
  • <script>: Embeds or references JavaScript code.
    <script>alert('Hello, World!');</script>
